# My Portfolio

## Description

This website allows an employer to view my portfolio using React.  No challenges were faced in creating this website.  In the future, I would like to make this website more visually pleasing.

## Installation

To view this website, one must first install React.  

## Usage

A potential employer can view this website to make an informed decision of my employment at their company.
